# Konings-IBM-Capstone-Project
From March to July of 2026 I had been taking the IBM Data Analyts Professional Certification on Coursera. It's was an interesting and engaging journey where I learnt more than I could imagine on Data Analysis.
This program has courses on Python, Excel, SQL, Generative AI for analysis as well as preparing youfor interciews into analytic roles.
This Repo contains what I did during the Capstone Project of the course. I used Python with its vast libraries and capabilities throughout as well as SQL, Excel, Cognos Analytics and more. 
The data was gotten from the Stack Overflow Developer Survey.
It started with data collection vis API and Web Scrapping techniques, went on to Data Wrangling to make the data analysis ready in which I handled duplicates, missing values, normalised data and much more.
I then went on to carrying out the actual analysis where I looked for trends and relationships with the data I was working with. I had to use various python functions such as groupby(), explode(), as well as the data structures I had picked up earlier in the course such as lists, dictionaries, etc.
I went on to visualise my findings using various charts/graphs such as Histograms, Bar charts, Line plots, Bubble charts, Pie charts and much more. I utilised Pandas, Matplotlib and seaborn for my visualisations.
Then, I created dashboards using IBM's Cognos Analytics software which helped me to put all my findings in one place and actually understand what each chart and interaction actually meant.
I ended up by creating a PowerPoint presentation summarising all my findings in the way professionals do. I have to say this was a bit tricky because I put myslef uder pressure to deliver like I would if I was faced with stakeholders. I had to make the report comprehensible to anyone it came across.
All in all this project was an amazing experience and a true eye-opener into what data analysis is all about. I had the opportunity to learn new things as well as strengthen my knowledge in Pyhton, Excel and SQL. There was never a dull moment as I was always on my feet, constantly being pushed to learn even during the project phase of it.
